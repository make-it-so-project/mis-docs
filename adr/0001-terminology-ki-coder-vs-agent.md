# ADR-0001 Terminology: KI-Coder vs Agent

## Status
Accepted

## Context

Im Projekt make-it-so existieren zwei unterschiedliche Bedeutungen des Begriffs Agent:
1. Entwicklungsdomäne: KI-gestützte Entwicklungseinheiten
2. Betriebsdomäne: fachliche Agenten innerhalb des Produkts

## Decision

Für die Entwicklungsdomäne wird der Begriff **KI-Coder** verwendet.
Für die Betriebsdomäne bleibt der Begriff **Agent** bestehen.

## Consequences

Begriffe in Architektur, Governance, Backlog und technischer Dokumentation werden klar getrennt.
Missverständnisse in Prompts, ADRs und Issues werden reduziert.
