# Glossar

## Begriffe

**KI-Coder**  
Begriff für spezialisierte KI-Entwicklungseinheiten in der Entwicklungsdomäne.

**Agent**  
Begriff für generische Agenten in der Betriebsdomäne von make-it-so.

## Backlog

**G-BL**  
Gesamt-Backlog. Raum für Ideen, Refinement und Zerlegung. Nicht ausführend.

**S-BL**  
Sprint-Backlog. Vom CEO freigegebener Ausführungsraum für KI-Coder.

## Entwicklung

**PR**  
Pull Request. Vorschlag, Änderungen aus einem Branch in einen Ziel-Branch zu übernehmen.

**ADR**  
Architecture Decision Record. Dokumentierte Architekturentscheidung.
